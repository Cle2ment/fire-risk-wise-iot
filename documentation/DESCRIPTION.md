# 基于AIoT的城市老旧小区智慧安防与消防通道联动管控系统

本课题聚焦智慧城市基层治理场景，针对老旧小区普遍存在的安防巡查效率低、消防通道违规占用难根治、高空抛物追溯难、异常险情响应滞后等民生痛点，构建轻量化AIoT智慧管控方案，方案无需大规模改造现有设施，部署成本低、适配性强。

预期技术应用：感知层依托部署在小区关键点位的高清摄像头、地磁车辆检测器、烟感温感传感器，通过研华工业边缘网关完成多源数据接入与边缘预处理，降低云端传输带宽压力；AI算法层在边缘端部署轻量化目标检测模型，精准识别消防通道违停车辆、通道堆物、高空抛物、异常人员徘徊四类风险事件，结合人体姿态识别算法实现老人摔倒等突发险情检测，所有识别运算均在本地完成，保障社区数据隐私；管控层搭建可视化智慧社区平台，通过规则引擎自动触发声光告警、物业工单派单，支持与街道智慧城市治理平台分级对接，实现事件从发现到处置的全流程闭环管理。

预期结果：可替代80%以上的人工定点巡查工作量，消防通道违规占用问题的整改处置效率提升60%；方案兼容小区现有监控与传感设备，边缘端部署模式无需高额算力投入，可快速适配不同规模的老旧小区，为城市社区智慧化升级提供可行方案。
